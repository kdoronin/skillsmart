Пример №1:
Класс для отправки данных с сервера в Google Analytics.
Необходимы статические методы, которые позволят инициировать один из типов событий, в зависимости от контекста.

Пример №2:
Класс для отправки попапов на фронтенд через REST API.
Имеет статические методы, генерирующие JSON для того или иного типа попапов.

Пример №3:
Класс формирующий данные для календаря событий.
Статические методы в данном классе готовят данные для основного класса, собирающего все данные и отправляющего их клиенту.


Интерфейсы и абстрактные классы, похоже, буду приводить в пример при следующем прохождении курса.
